## 유용하게 쓰는 명령어들

| 명령어                     | 설명             |
| -------------------------- | ---------------- |
| source /etc/profile        | profile 적용     |
| free -h                    | 메모리 용량 확인 |
| dh -h                    | 디스크 용량 확인 |
| nproc                      | CPU 코어 수 확인 |
| pwd                        | 현재 경로 확인   |
| chmod -R 755 test | 사용권한 변경    |
| chown -R root:root test | 소유자와 그룹 변경   |
| ps -ef &#124; grep '프로세스명' | 프로세스 확인   |

## 가끔 쓰인 명령어들

| -------- | -------------- |
| uname -r | 커널 버전 확인 |
