## 자주 썼던 명령어들

| 명령어              | 설명             |
| ------------------- | ---------------- |
| source /etc/profile | profile 적용     |
| free -h             | 메모리 용량 확인 |
| nproc               | CPU 코어 수 확인 |

## 자주 쓰지 않았던 명령어들

| 명령어   | 설명           |
| -------- | -------------- |
| uname -r | 커널 버전 확인 |
